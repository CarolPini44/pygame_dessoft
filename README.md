# TITULO DO PROJETO: 
Hero Rock Fox

# NOME DOS INTEGRANTES: 
Carolina Pini, Gabriel Rodrigues, Ana Luiza Alves

# O JOGO:
O Hero Rock Fox é um jogo feito no pygame e inspirado no Guitar Hero, do gênero de jogos de ritmo. O objetivo do jogador é selecionar uma das músicas disponíveis para que o mesmo acerte as notas que caem de acordo com o ritmo da música tocada. 
# COMO JOGAR:
O jogo utiliza quatro pistas para representar diferentes notas com cores diferentes. Na tela incial o qual aparece quando o jogador acaba de iniciar o programa, utiliza-se a barra de espaço para ir ao menu de seleção.
No menu de seleção, as teclas escolhidas para a interação foram "ESC" para voltar para a tela inicial, setas para cima e para baixo que auxilia o jogador na escolha das músicas e a barra de espaço para selecionar a música escolhida.
Na tela de partida, para o acerto de notas musicais, foram escolhidas as teclas "A" para representar a nota de cor azul, "S" para a de cor vermelho, "K" para a cor ver e "L" para a cor amarela. Para o jogador sair para a tela final do jogo, utiliza-se a barra de espaço para mostrar a quantidade de pontos que o jogador conseguiu ao longo da partida.
Na tela final, também é mostrado ao jogador as teclas de interação "ESC" para voltar ao menu e barra de espaço para fechar o jogo.

# ESTRUTURA DO CÓDIGO:
Para executar o programa, basta rodas o arquivo "main.py" da pasta.
Foram utilizadas as bibliotecas do pygame, os, sys e random (apenas no código mas não utilizada).
Foi utilizado na parte das listas de cada nota IA para separa-las de forma organizada e nos tempos certos das batidas.
Foram feitas 2 sprites utilizáveis no código, sendo elas "NotaCaindo" e "NotaJulgamento".
